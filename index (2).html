Configura sync cloud
Il sync usa un gist privato GitHub come deposito dei tuoi dati. Niente di ciò che inserisci qui esce dal tuo account GitHub.
  1. Apri questo link — è la pagina GitHub per creare un Personal Access Token classico, già pre-compilata con scope gist (l'unico necessario).
  2. Imposta una scadenza (suggerito: 1 anno) e premi Generate token in fondo.
  3. Copia il token (inizia con ghp_) e incollalo qui sotto.
Il token viene salvato solo nel localStorage di questo dispositivo. Se vuoi disattivare il sync, basta premere "Disconnetti" (o revocare il token da GitHub).
Se hai già usato il sync su un altro dispositivo, lo stesso token connetterà tutti automaticamente: l'app trova il gist esistente e tira giù i dati.
